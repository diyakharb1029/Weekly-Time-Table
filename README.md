# Weekly Time Table

A simple weekly timetable layout built using HTML and CSS to practice basic web layout and styling concepts.

## Tech Stack
- HTML
- CSS

## About
This is a beginner-level front-end project focused on creating a clean, readable, and structured weekly timetable using HTML tables and basic CSS styling.
